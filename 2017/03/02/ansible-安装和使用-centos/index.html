2017-03-02

ansible, centos

  1. 安装

    需要先安装扩展源EPEL
    sudo yum -y install epel-release

    安装ansible
    通过Yum安装最新发布版本
    通过Yum安装RPMs适用于 EPEL 6, 7, 以及仍在支持中的Fedora发行版.
    托管节点的操作系统版本可以是更早的版本(如 EL5), 但必须安装 Python 2.4 或更高版本的Python.
    Fedora 用户可直接安装Ansible, 但RHEL或CentOS用户,需要 配置 EPEL

    $ sudo yum install ansible
